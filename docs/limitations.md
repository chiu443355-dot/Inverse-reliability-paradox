# Research Limitations

This framework remains exploratory and observational.

The project does NOT claim:
- universal causation
- production deployment validity
- direct operational certification

---

# Main Constraints

## Dataset Constraints
- public datasets
- incomplete operational telemetry
- proxy congestion variables

## Simulation Constraints
- synthetic overload modeling
- probabilistic assumptions
- simplified queue structures

## Infrastructure Constraints
- no live warehouse integration
- no IoT telemetry
- no enterprise-scale deployment
